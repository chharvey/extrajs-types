path

  • get path(): Integer[]

  • Inherited from TreeNode.path

    • Defined in TreeNode.class.ts:160

    Return an array of indices describing the path of this TreeNode in its tree.

    If this node has no parent, an empty array is returned.

    Each index in the array describes the position of an ancestor of this node, where the order of indices begins with the root and ends with this node. That is, the first entry is the index of this node’s ancestor within the root’s children; the second entry is the index of this node’s ancestor within the root’s grandchildren; and so on.

    To get the “depth” of a node, you can call node.path.length.

    For example, this diagram shows a tree with nodes. The paths of the nodes are listed below it.

           r (root)
      / \
     a   b
    /     \
   /\      e
  c  d     |
           f
    r.path  //  []  (the path of a root node is an empty array)
a.path  //  [0]
b.path  //  [1]
c.path  //  [0, 0]
d.path  //  [0, 1]
e.path  //  [1, 0]
f.path  //  [1, 0, 0]

    NOTE: The returned array is shallow, not live, meaning any changes to it will not affect the tree.

    Returns Integer[]

    an array of non-negative integers describing the position of each of this TreeNode’s ancestors
